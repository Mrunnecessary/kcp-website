\# Kolkata Cocoa Parlor 🍫



Handmade chocolate gifting website with automated order system.



\## 🚀 Features

\- Online order form

\- Google Sheets database

\- Email automation (Apps Script)

\- WhatsApp integration

\- PAN India delivery



\## 🧠 Tech Stack

\- Frontend: HTML (Vercel)

\- Backend: Google Apps Script

\- Database: Google Sheets



\## 🔁 Order Flow

Website → Form → Google Sheet → Email → Payment → Delivery
